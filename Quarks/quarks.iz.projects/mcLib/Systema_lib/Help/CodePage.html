CodePage converts UTF8 keyboard input into custom extended-ascii encodings


Inherits from: Object


Keyboard input on a macintosh can be converted into a limited set of extended-ascii charaters. Specially designed fonts display these characters in respect of the 'CodePages' defined in the config file.
