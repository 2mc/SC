Strange shapes that result from letting one parameter "plex" take successively values from the arithmetic series 1, 2, 3 ...


Here showing "plex" values from 1 to 10, and realizing each time 2 variants: one with 25 iterations and 50 iterations. 



*/



(

f = { | plex = 1, iterations = 25 |

Turtle.draw({ | t |

var metagonSide = 64;

Pseq({ | i |

Pfuncn({ t.turn(2 ** (i * plex) * pi / (i + 1 ** 1.001)); t.move(metagonSide, 1) }, i + 2)

} ! iterations

, 1)

},  x: 400, y: 400,rate: 0);

};


r = {

(1..10) do: { | plex |

[25, 50] do: { | iterations |

f.(plex, iterations);

8.wait;

}

}


}.fork(AppClock);


)



// variants exploring values of plex (1..10) / 10 as well as (1..10) / 100
