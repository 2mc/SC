TUIO


Dependencies


This quark requires the ServerPrep quark from iz.core Quarks group.


Data Format of received TUIO messages for each tracked blob (session)



  /tuio/2Dcur set s_id x_pos y_pos x_vel y_vel m_accel



Usage



TUIO.new; // create and activate a new session


TUIO.deactivate // deactivate the default TouchOSC instance


To customize the behavior of TouchOSC, store functions to the following instance variables: 


sessionCreated

sessionChanged

sessionEnded


Like this: 


TUIO.new.sessionCreatedAction = { | session |

session.id.postln;

// \default.play;

// { WhiteNoise.ar }.play;

// { SinOsc.ar(400 + 1000.rand, 0, EnvGen.kr(Env.perc(level: 0.1), doneAction: 2)) }.play;

Synth("test");

};




TUIO.new.sessionCreatedAction = { | session |

// session.id.postln;

{ WhiteNoise.ar(EnvGen.kr(Env.perc(level: 0.1), doneAction: 2)) }.play;

};



Udef("test", { 

var source, env;

env = Env.perc;

source = SinOsc.ar(Rand(400, 4000), 0, 0.1);

Out.ar([0, 1], source * EnvGen.kr(env, doneAction: 2));

});


Synth("test");
