WT_SoftThreshold softly threshold coefficients


WT_SoftThreshold(buffer, threshold = 0.0, startindex = 1)


Zeroes out any coefficients whose magnitude is not above a threshold. Those above are mapped to: 

coeff - sgn(coeff)*threshold

Which is a softer effect rather than hard thresholding, though it reduces amplitude in the resonstructed signal still further. 


buffer - dwt buffer.

threshold - absolute magnitude threshold for coefficients

startindex - index of coefficient to begin thresholding at. Defaults to 1 to avoid zeroing the maximal lowpass smoothing information before reconstruction, but you can threshold on that too if you set this to 0 (or start at a higher scale). 




s.boot;


b = Buffer.alloc(s,1024,1)


(

SynthDef("softthreshold", { arg out=0;

var in, chain;

in = WhiteNoise.ar(0.2);

//in = SoundIn.ar; 

chain = DWT(b, in);

chain = WT_SoftThreshold(chain, MouseX.kr(0.0,0.5)); 

//chain.poll;

Out.ar(out, 0.5 * IDWT(chain).dup);

}).play;

)
