
INSTALLATION INSTRUCTIONS

The present library is installed in SuperCollider using RepQuarks. To install it:

1. Make an alias of the folder "PutMyAliasInExtensions" and put it in the Extensions folder of your Platform.userAppSupportDir
2. Recompile SuperCollider
3. Choose any quarks you want to include from the Quarks menu. 
4. Press the save button. 
5. Recompile the SuperCollider library when the message on the Quarks window tells you to.
