SynthLink


Inherits from: Object

Purpose


Encapsulate a Function that creates a UniqueSynth so that it can be evaluated within the environment of a Chain. 


Eplanation


Functions that create UniqueSynths can be passed to a Chain directly, for example: 


(

Chain(Pseq([

{ \default.play([\freq, 200]).dur(0.1, 0.1); },

{ { Resonz.ar(WhiteNoise.ar(5), 1000, 0.01) }.play.dur(0.1, 0.1); },

{ { | freq = 400 | SinOsc.ar(freq, 0, 0.1) }.play(args: [\freq, 500]).dur(0.1, 0.1); },

], 20));

)


However, for all synth functions of a chain to share data between them, these need to be stored in the environment of the chain, which is passed to each function in the chain when it is played. Thus, to share patterns or other elements between the functions in a chain, one must store the synth-making function in an object that will evaluate it in the environment of the Chain. That task is performed by SynthLink. To store a Function in a SynthLink, sending the message s to the Function:


(

Chain(Pseq([

/* note: ~freq.next and \freq.n are equivalent */

{ \default.play([\freq, ~freq.next]).dur(~dur.next, ~fade.next); },

{ { Resonz.ar(WhiteNoise.ar(5), \freq.n, 0.01) }.play.dur(\dur.n, \fade.n); },

{ { | freq = 400 | SinOsc.ar(freq, 0, 0.1) }.play(args: [\freq, \freq.n]).dur(\dur.n, \fade.n); },

], 20),

() make: { // store the shared patterns in the global environment:

\dur.pseq([0.1, 0.2], inf);

\fade.pseq([0.1, 0.2, 1], inf); 

\freq.pseq([80, 85, 87, 90, 92].midicps, inf) 

});

)



Usage


A SynthLink instance is created from a Function that returns a UniqueSynth instance when evaluated, by sending that function the message s.
