Chain


Inherits from: Object

Purpose


Play a pattern producing fuctions whose values are objects that respond to onEnd (UniqueSynths, EventStreams), by chaining each next element so as to start playing when the previous element ends. 


Usage


//:Chain example

Chain(Prand([

{

{

\default.mplay([\freq, (50..80).choose.midicps]).dur(0.03, exprand(0.01, 0.3));

\hocketus.stream(Prand([0.06, 0.07, 0.14], 10 rrand: 20));

}.stream;

},

{ { | freq = 400 | SinOsc.ar(freq * [1, 1.2], 0, 0.2) }

.play(args: [\freq,  \freq.pseq((4..20), inf) * 100]).dur(0.1 rrand: 1, 0.5 rrand: 2.5) 

}

],

30

));


//:The nested function of the first element in chain above can be simplified as: 

Chain(Prand([

{

\default.mplay([\freq, (50..80).choose.midicps]).dur(0.03, exprand(0.01, 0.3));

\hocketus.stream(Prand([0.06, 0.07, 0.14], 10 rrand: 20));

}.chain,

{ { | freq = 400 | SinOsc.ar(freq * [1, 1.2], 0, 0.2) }

.play(args: [\freq,  \freq.pseq((4..20), inf) * 100]).dur(0.1 rrand: 1, 0.5 rrand: 2.5) 

}

],

30

));


//:Second chain example

Chain(Prand([

chain { // chain message can also been written before the function like in loop, while etc.

{ Klang.ar(`[ Array.rand(3, 100.0, 125.0) * \inc.prand((5..7), inf)]) * 0.05 }

.play.dur(0.1, 0.3);

\dur.pn(0.1, 5);

},

{ { | freq = 400 | SinOsc.ar(freq * [1, 1.1], 0, 0.2) }

.play(args: [\freq,  \freq.pseq((1..25), inf) * 100]).dur(0.01 rrand: 0.1, 0.05 rrand: 0.5) 

}

],

35

));


//:Second chain example with Pseq instead of Prand

Chain(Pseq([

chain { // chain message can also been written before the function like in loop, while etc.

{ Klang.ar(`[ Array.rand(3, 100.0, 125.0) * \inc.prand((5..7), inf)]) * 0.05 }

.play.dur(0.1, 0.3);

\dur.pn(0.1, 5);

},

{ { | freq = 400 | SinOsc.ar(freq * [1, 1.1], 0, 0.2) }

.play(args: [\freq,  \freq.pseq((1..25), inf) * 100]).dur(0.01 rrand: 0.1, 0.05 rrand: 0.5) 

}

],

25

));


//: End second chain example


Using chain to send sequences of OSC messages. 


//:Third chain example

Chain(Prand([

{

NetAddr.localAddr.sendMsg(*[\test1, \count.pseries(101)].postln);

\default.mplay([\freq, 30.rrand(100).midicps]).dur(0.1, 0.1);

}.chain2({ \dur.pn(0.1, 5) });,

{

NetAddr.localAddr.sendMsg(*[\test2, \count.pseries(1) * 10].postln);

}.chain2({ \dur.pn(1, 3) });

],

30

)

);

//: End third chain example

Instance Creation


*new(pattern, envir) : Create a new instance of Chain and start it

pattern: A pattern producing functions that when evaluated create objects that respond to onEnd.

envir: an Environment or Event.

The functions are evaluated using envir as environment. Default is an empty event.
