Spectrograph


Inherits from: UniqueWindow : UniqueObject : Object

 

A real time Spectrogram  with customizable window dimensions and colors. 


//:Customize Spectrograph background color

Spectrograph.backgroundColor = Color(0.2, 0.5, 0.9);

//:Start Spectrograph with some test tones.

Spectrograph.start;
