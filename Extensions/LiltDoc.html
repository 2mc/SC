LiltDoc provides a way to execute parts of the code in a SuperCollider document through keyboard commands. 


It only works with documents in plain text, of the form <filename>.scd


To mark part of a document for execution by LiltDoc, precede it by a comment in the form of //:


Example: 


//:1 Start Sound 1


a = { PinkNoise.ar(0.1) }.play;


//:2 Stop Sound 1


a.free;


//:Rest 


The code above defines two "code snippets" and binds them to (keyboard) keys 1 and 2. 


Activation of LiltDoc keyboard commands on a document


The current version is being developed on MacOS, so it uses the user menu to install a global keyboard shortcut. 




To activate LiltDoc keyboard commands on the current document being edited, type Command-v.
