BMSoundFilePlayer a sound file player


Inherits from: Object : BMAbstractAudioChainElement : BMAbstractAudioSource


BMSoundFilePlayer is a soundfile player with integration in the BEASTmulch library. It provides standard functionality, i.e. play, pause, stop, change rate, etc.


Instances can serve as Time References, and will update any dependants with update(\time, timeInSeconds, rateOfChange, referenceTime) messages. (See class Object for a discussion of dependancy mechanisms.) It also updates dependants with the following messages:


\loading - when loading of a file begins

\loaded - when a sound file has been loaded

\base - when its time base has changed (i.e. a file has been loaded or buffer cleared)

\play - when playback begins or resumes

\playFailed - if playback failed

\stop - when playback stopped

\pause - when playback pauses

\bufferFreed - when its buffer is freed


See also: BMAbstractAudioSource, BMSoundFilePlayerGUI, Time References and Automation


File Size Limits


In the current implementation BMSoundFilePlayer loads files into memory rather than streaming from disk. This avoids potential dropout, etc. resulting from disk access, but places some practical upper limits on file size.


Creation / Class Methods


*new (maxNumChannels, target, addAction, name)

Create a new BMSoundFilePlayer.

maxNumChannels - The maximum number of output channels for this player. Default value is 2.

target - Target for this object's Group. If target is not a Group or Synth, it will be converted as follows: If it is a Server, it will be converted to the default_group of that server. If it is nil, to the default group of the default Server. Note: A Synth is not a valid target for \addToHead and \addToTail.

addAction - one of the following Symbols:

\addToHead - (the default) add at the head of the group specified by target

\addToTail - add at the tail of the group specified by target

\addAfter - add immediately after target in its server's node order

\addBefore - add immediately before target in its server's node order

\addReplace - replace target and take its place in its server's node order

name - A Symbol indicating the name of this object. If not supplied one will be generated.



Accessing Instance Variables

sampleDur

Returns the sample duration based on the currently loaded buffer's sample rate.

buffer

Get the currently loaded Buffer.

rate_

rate

Set or get the current playback rate. 1 is normal, -1 is backwards, 2 is twice as fast, etc.

loading

Returns a Boolean indicating if a file load is in progress.

synth

If playing, return this object's playback Synth, nil otherwise.

bus_

bus

Set or get this object's output audio Bus. This should match maxNumChannels in size. 

Instance Methods


read (path, action)

Read a sound file into memory.

path - A String indicating the path of the file to be loaded.

action - An optional Function to be evaluated when loading is complete. It will be passed the receiver as an argument.

p = BMSoundFilePlayer(name:\foo);

p.read("sounds/a11wlk01.wav", {|pl| "% read successfully\n".postf(pl.name)}); 

p.free; // cleanup



play (startTime, out)

Play the currently loaded file. If playing is already underway this is ignored. At the end of the file playback will stop automatically.

startTime - Offset in seconds into the file at which to start playback.

out - An optional Integer or Bus to use for output. If nil (the default) the object's internal (private) bus is used.

p = BMSoundFilePlayer(name:\foo);

p.read("sounds/a11wlk01.wav");

p.play(0, 0); // play from start out bus 0;

p.play(1, 0); // skip ahead

p.play(0, 0).rate_(0.5); // half speed

p.free;



stop

If currently playing stop, otherwise ignore.



togglePlay

If currently playing, pause playback. If stopped or paused, start or resume playback.



pause

If currently playing pause playback.



freeBuffer

Free the currently loaded Buffer.

free

Free this object's resources and nodes. After calling this it should not be used.



setTime (time)

If currently playing jump to the new time, if not start playback at time in a paused (i.e. cued) state.

time - The new time in seconds.



asBMInOutArray

Returns this object's outputs as a BMInOutArray.

p = BMSoundFilePlayer(name:\foo);

p.asBMInOutArray; // foo-1 etc.

p.free;




path

Returns a String indicating the path of the currently loaded sound file.



gui

Creates and returns a BMSoundFilePlayerGUI with this object as an argument.

p = BMSoundFilePlayer(name:\foo);

p.bus = 0; // for the sake of example

p.gui.onClose_({p.free});



Examples


// example with a matrix for routing

s.boot;


p = BMSoundFilePlayer(name:\foo);

p.read("sounds/a11wlk01.wav"); // mono, so only out foo-1


// make a speaker list

(

l = [[-30, 'L'], [30, 'R'], [0, 'C'], [-110, 'Ls'], [110, 'Rs']].collectAs({|el, i|

BMSpeaker.newFromSpherical(el[1], i, el[0], 0, 2);

}, BMInOutArray);

)


// make a matrix for connections

m = BMAudioMatrix(p, l, name:'Router');


m.connect('foo-1', 'L');


m.gui; p.gui; // play with the GUIs

p.free;
