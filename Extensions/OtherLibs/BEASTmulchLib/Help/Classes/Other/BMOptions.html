BMOptions stores some global options for the BEASTmulch library


Inherits from: Object


BMOptions is a utility class which stores some global options and information for the BEASTmulch library.


Class Methods


*defaultServerOptions

Returns a new instance of ServerOptions using the server related class vars detailed below. These are tweaked for use with the library and may form a more practical starting point than the values found in ServerOptions itself.



General Class Variables

*version

Returns an Array in the form [majorVersion, minorVersion, subVersion]. Thus 1.0.0 is [1, 0, 0].

*crossfade_

*crossfade

Set or get the default crossfade time in seconds used throughout the library, for example whan adding or removing connections in matrix classes. Default value is 0.1.

*allowMultipleControlMappings_

*allowMultipleControlMappings

Set or get a Boolean indicating whether BMControl instances can be mapped to more than one object at once. Default value is false.

Class Variables for *defaultServerOptions


The following setters and getters determine the values set in *defaultServerOptions. See ServerOptions for details on their meaning.


*numInputBusChannels_

*numInputBusChannels

Default value is 8.

*numOutputBusChannels_

*numOutputBusChannels

Default value is 8.

*numWireBufs_

*numWireBufs

Default value is 512.

*numAudioBusChannels_

*numAudioBusChannels

Default value is 1024.

*memSize_

*memSize

Default value is 131072.

Examples


// using defaultServerOptions

o = BMOptions.defaultServerOptions;

o.blockSize = 128; // vary one of the options

s.options = o;

s.boot;
