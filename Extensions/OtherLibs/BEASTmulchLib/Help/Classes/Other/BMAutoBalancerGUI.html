BMAutoBalancerGUI a GUI for BMAutoBalancer


Inherits from: Object : BMAbstractGUI


This class provides a GUI for the utility class methods found in BMAutoBalancer. Hardware input channels start from 1 in the GUI, not 0.


See also: BMAbstractGUI, BMAutoBalancer


Creation / Class Methods


*new (speakerList, okayFunc, server, name, origin)

Create a new BMAutoBalancerGUI.

speakerList - The BMInOutArray to balance. This is done in place.

okayFunc - An optional Function to evaluate when the cycle is complete. This will be passed speakerList as an argument.

server - The Server to use. nil indicates the default server.

name - An optional String or Symbol indicating the title of the window. Default is "Autobalance Speakers".

origin - An optional Point specifying the origin of the window. Default is 400@600.



Examples
