BMSoundFilePlayerGUI GUI class for BMSoundFilePlayer


Inherits from: Object : BMAbstractGUI


BMSoundFilePlayerGUI creates a gui for a BMSoundFilePlayer. It provides controls for playback, loading and clearing buffers, and an optional 'Big Clock'. (Click the clock icon.)


See also: BMAbstractGUI, BMSoundFilePlayer


Creation / Class Methods


*new (player, name)

Create a new BMSoundFilePlayerGUI.

player - A BMSoundFilePlayer.

name - An optional String or Symbol indicating the title of the window. If nil is passed the name of player will be used.



Instance Methods


makeBigClock

Open the 'Big Clock' auxiliary window, if not already open. You can also do this by clicking on the clock icon.



Examples


p = BMSoundFilePlayer(2, name:\foo); // stereo player

p.bus = 0; // for the sake of example

g = BMSoundFilePlayerGUI(p).onClose_({p.free}); // you'll need to load a file

g.makeBigClock // can also click on the clock icon for this
