BMSnapShotSeq a sequence of automation snapshots


Inherits from: Object


BMSnapShotSeq is a time ordered sequence of automation snapshots. Under normal circumstances you will not need to make these directly, but will rather create them through an automator object, for example an instance of BMControllerAutomator.


See also: BMAbstractSnapShot, BMArbitraryStartSnapShot, BMSnapShotSeq, BMAbstractAutomator, BMControl, BMAbstractController, BMSnapShotSliders, Time References and Automation


Creation / Class Methods


*new (name, controls, firstSnapTime, curve)

Create a new snapshot sequence.

name - A Symbol indicating the name of this snapshot sequence.

controls - An Array of Symbols corresponding the names of the BMControl instances to be automated.

firstSnapTime - A time in seconds corresponding to the start time of this sequence. An initial snapshot will be made of the values of the controls and added at this time, and an arbitrary snapshot added 1 second before this time.

curve - A curve to use for interpolation segments. Valid values are the same as those for Env. The default is 'lin'.



Accessing Instance and Class Variables

name

Returns a Symbol indicating the name of this snapshot sequence.

snapshots

Returns an Array containing this sequence's snapshots in time order.

start

Returns the time of the first snapshot in this sequence in seconds. Since the first snapshot is arbitrary, this may be earlier than the firstSnapTime specified above.

end

Returns the time of the last snapshot in this sequence in seconds.

curve

Returns the curve used for interpolation segments. This will be one of those specified in the help file for Env.

snapshotsDict

Returns an IdentityDictionary in the form [snapshot1Name->snapshot1, snapshot2Name->snapshot2].

minSegSize_ (newSize)

minSegSize

Set or get the minimum size for an interpolation segment in seconds. This can be useful in preventing physical controllers from being overloaded.

duration

Returns the duration of this sequence in seconds.

Instance Methods



addSnapShot (time, ssname)

Add a new snapshot to this sequence. This will capture the current state of the controls.

time - The time at which to add this snapshot. This will be relative to a Time Reference. See Time References and Automation.

ssname - A Symbol indicating the name of this snapshot.



removeSnapShot (ssname)

Remove a snapshot from this sequence.

ssname - A Symbol indicating the name of the snapshot to remove.



atTime (time)

Returns an IdentityDictionary in the form [ctrlname->valueAtTime, ...]. 

time - Explanation of time. Default value is nil. Other information.



containsTime (time)

Returns a Boolean indicating whether time is inclusively between this sequences start and end times.

time - The time in seconds to test.
