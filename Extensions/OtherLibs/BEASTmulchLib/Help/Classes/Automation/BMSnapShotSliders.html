BMSnapShotSliders a GUI for showing and editing the state of a BMSnapShot


Inherits from: Object : BMAbstractGUI


BMSnapShotSliders is a modal sheet for showing and editing the state of a BMSnapShot. This uses BMControl:displaySpec to determine the correct range and units for display based on what the control is mapped to. Normally you would not need to create one of these directly, but it is used by BMControllerAutomatorGUI.


See also: BMSnapShot, BMControllerAutomatorGUI, BMControllerAutomator


Creation / Class Methods


*new (snapshot, parent)

snapshot - The BMSnapShot whose state is to be represented.

parent - The parent window. Usually this would be the window of a BMControllerAutomatorGUI.
