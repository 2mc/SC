BMSnapShot represents an automation snapshot


Inherits from: Object : BMAbstractSnapShot


BMSnapShot represents a snapshot in an automation sequence. Its public methods are all defined in its superclass.


Under normal circumstances you will not need to make these directly, but will rather create them through a BMSnapShotSeq.
