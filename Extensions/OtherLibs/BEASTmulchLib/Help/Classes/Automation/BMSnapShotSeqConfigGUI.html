BMSnapShotSeqConfigGUI a GUI to select controllers for a snapshot seq


Inherits from: Object : BMAbstractGUI


BMSnapShotSeqConfigGUI provides a modal sheet to select controls for a new snapshot sequence. Normally you would not create one directly, but it is used by BMControllerAutomatorGUI.


See also: BMSnapShot, BMControllerAutomatorGUI, BMControllerAutomator


Creation / Class Methods


*new (parent)

parent - The parent window. Usually this would be the window of a BMControllerAutomatorGUI.



Examples


(

c = BMVirtualController(\foo); // make some controls to select

BMSnapShotSeqConfigGUI(SCWindow("Fake").front).onClose_(_.postln);

)


c.free;
