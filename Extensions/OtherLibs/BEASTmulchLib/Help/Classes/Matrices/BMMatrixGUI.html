BMMatrixGUI a GUI for matrix classes


Inherits from: Object : BMAbstractGUI


BMMatrixGUI provides a GUI for subclasses of BMAbstractMatrix. Inputs are represented on the vertical axis and outputs on the horizontal axis. BMMatrixMenuGUI provides an alternative view.


See also: BMAbstractMatrix, BMAbstractGUI, BMMatrixMenuGUI


Creation / Class Methods


*new (matrix, name)

Create a new matrix GUI.

matrix - An instance of a subclass of BMAbstractMatrix.

name - An optional String or Symbol indicating the title of the window. If nil is passed the name of matrix will be used.



Examples


(

// make a speaker list

l = [[-30, 'L'], [30, 'R'], [0, 'C'], [-110, 'Ls'], [110, 'Rs']].collectAs({|el, i|

BMSpeaker.newFromSpherical(el[1], i, el[0], 0, 2);

}, BMInOutArray);

)


// something to route

(

b = Bus.audio(s, 8);

x = { Out.ar(b, {|i| Ringz.ar(Dust.ar(3, 0.1), 300 * (i + 1), 2)} ! 8) }.play;

)


// make a matrix

m = BMAudioMatrix(b.asBMInOutArray(\ringzes), l, name: 'Routing');


s.meter; // so we can watch
