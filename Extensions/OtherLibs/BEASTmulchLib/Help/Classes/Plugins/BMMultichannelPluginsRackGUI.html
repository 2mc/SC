BMMultichannelPluginsRackGUI GUI class for a BMMultichannelPluginsRack


Inherits from: Object : BMAbstractGUI


BMMultichannelPluginsRackGUI is a GUI class for control of a BMMultichannelPluginsRack. If provides facility for plugin creation, and control.


See also: BMMultichannelPluginsRack, BMMultichannelPlugin


Creation / Class Methods


*new (trimPluginsRack, name, origin)

Create a new BMMultichannelPluginsRackGUI.

pluginsRack - An instance of BMMultichannelPluginsRack.

name - An optional String or Symbol indicating the title of the window. If nil is passed the name of pluginsRack will be used.

origin - An optional Point specifying the origin of the window. Default is 40@200.



Examples


s.boot;


// 5.1 Array (-0.1)

(

l = [[-30, 'L'], [30, 'R'], [0, 'C'], [-110, 'Ls'], [110, 'Rs']].collectAs({|el, i|

BMSpeaker.newFromSpherical(el[1], i, el[0], 0, 2);

}, BMInOutArray);

)


b = Bus.audio(s, 1);


x = {Out.ar(b, PinkNoise.ar(0.2))}.play; // something to pan


i = b.asBMInOutArray(\noiseBus);


r = BMMultichannelPluginsRack(i, l, x, \addAfter, 'Plug em In');


r.addPlugin(BMMultichannelPlugin('2D VBAP Auto Panner', i, l));


s.meter;


BMMultichannelPluginsRackGUI(r); // play with the GUI


x.free; r.free; b.free;
