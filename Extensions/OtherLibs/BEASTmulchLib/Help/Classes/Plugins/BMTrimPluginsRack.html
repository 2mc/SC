BMTrimPluginsRack manages trim and plugins on a channel


Inherits from: Object : BMAbstractAudioChainElement


BMTrimPluginsRack manages trim settings and plugins (instances of BMPlugin) for its input channels. It can thus provide a processing stage in an audio chain.


The individual channels of a rack are manipulated using instances of BMTrimPluginsStrip.


See also: BMPlugin, BMPluginSpec, BMInOutArray, BMAbstractAudioChainElement, BMTrimPluginsRackGUI, BMTrimPluginsStrip, BMMultichannelPluginsRack


Creation / Class Methods


*new (ins, target, addAction, name)

Create a new BMTrimPluginsRack.

ins - An instance of BMInOutArray indicating the channels which are to be scaled and processed.

target - Target for this object's Group. If target is not a Group or Synth, it will be converted as follows: If it is a Server, it will be converted to the default_group of that server. If it is nil, to the default_group of the default Server. Note: A Synth is not a valid target for \addToHead and \addToTail.

addAction - one of the following Symbols:

\addToHead - add at the head of the group specified by target

\addToTail - (the default) add at the tail of the group specified by target

\addAfter - add immediately after target in its server's node order

\addBefore - add immediately before target in its server's node order

\addReplace - replace target and take its place in its server's node order

name - A Symbol indicating the name of this object. If not supplied one will be generated.



Instance Methods


mappings

Returns an IdentityDictionary containing the current values for each channel for this rack, in the form IdentityDictionary[\channelName->IdentityDictionary[trim->trimValue, plugins->[[plugin.spec.name, plugin.attributes, plugin.values], ...] ], ...]. Subclasses may override this to include additional parameters.



mappings_ (dict)

Set this objects trim and plugin settings.

dict - An IdentityDictionary containing the current values for each channel for this rack, in the form IdentityDictionary[\channelName->IdentityDictionary[trim->trimValue, plugins->[[plugin.spec.name, plugin.attributes, plugin.values], ...] ], ...]. Subclasses may override this to include additional parameters.



at (channel)

Returns the BMTrimPluginsStrip for the indicated channel. This can be used to manipulate values for individual channels.

channel - The name of the channel as a Symbol.



gui

Creates and returns a BMTrimPluginsRackGUI using this as an argument.



free

Free this strip's plugins, group and other resources. After this you should not use this object.



delayCompensateDistance(bool)

Enable or disable delay compensation for distance. This will check the location of all BMSpeaker instances in ins and add a delay plugin so that sound will arrive at {0,0,0} simultaneously for all speakers. This can be used to eliminate precedence effect.

bool - A Boolean value. The default is true.



autoPlugins(bool)

BMSpeakerSpec instances can optionally define plugins and settings which should be used with speakers based on them, e.g. LPF on a subwoofer. This method enables or disables them for all speakers in this racks ins.

bool - A Boolean value. The default is true.



autoTrim

BMSpeaker instances can have autoTrim values, for example determined using BMAutoBalancer. This method sets the trim settings to the autoTrim value for all channels in ins which output to speakers with defined autoTrim values. nil trim settings are ignored.



Examples


// 5.1 Array (-0.1)

(

l = [[-30, 'L'], [30, 'R'], [0, 'C'], [-110, 'Ls'], [110, 'Rs']].collectAs({|el, i|

BMSpeaker.newFromSpherical(el[1], i, el[0], 0, 2);

}, BMInOutArray);

)


r = BMTrimPluginsRack(l, name: 'Plug em In');


r[\L]; // get the BMTrimPluginsStrip for 'L'


// plugins are added at the end of the chain

r[\L].addPlugin(BMPlugin('3 Band EQ'));

r[\L].addPlugin(BMPlugin(\Highpass));

r[\L].addPlugin(BMPlugin(\Lowpass));

r[\L].plugins; // check order


// but you can move them

r[\L].movePluginUp(2);

r[\L].movePluginDown(0);

r[\L].plugins; // check order


// remove by index

r[\L].removePlugin(0);

r[\L].plugins;


r.gui; // play with the GUI


r.free;
