BMVirtualController generic software based controller


Inherits from: Object : BMAbstractController


BMVirtualController is a generic software based controller. It provides a back end model and could be represented by any number of GUIs. One example, BMVirtualControllerSliders, is provided in the library. 


See also: BMAbstractController, BMVirtualControllerSliders


Automation


This class is automatable, but when creating GUI classes to represent the state of this controller, it will likely be necessary to separate GUI updates from automation updates, since too high a GUI refresh rate will cause the system to hang (spinning beachball). BMVirtualControllerSliders provides one example of how to do this using a refresh loop which is stopped and started as necessary.


Creation / Class Methods


*new (name, server, numControls)

Create a new BMVirtualController.

name - A Symbol indicating the name of this controller.

server - An instance of Server. If you pass nil this will be the default Server.

numControls - An Integer indicating the number of controls this controller will have. The default value is 8.

c = BMVirtualController.new(\sliders, s, 4); // new controller

g = BMVirtualControllerSliders(c);


g.close; c.free; // free resources



The following three methods are useful when creating instances from GUIs.


*newFromParamDict (dict, server)

Create a new instance using a parameter dictionary. (See BMAbstractController for more information.)

dict - A Dictionary or subclass with values for the following keys: \numControls -> An Integer indicating the number of controls, name -> A Symbol indicating the name of this controller.

server - An instance of Server. If you pass nil this will be the default Server.



*parameterList

See BMAbstractController for a discussion of this method. This is the implementation for BMVirtualController:

*parameterList { 

var class;

class = this;

^(

name: [Symbol, {class.makeName}, "Name"],

numControls: [Integer, [1, 64, \linear, 1].asSpec, "Number of Faders"]

); 

}



*humanName

Returns the String "GUI Controller".
