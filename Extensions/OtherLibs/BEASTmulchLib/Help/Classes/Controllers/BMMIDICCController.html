BMMIDICCController Generic MIDI Control Change Controller


Inherits from: Object : BMAbstractController : BMAbstractMIDIController


This class implements a generic controller which receives MIDI control change messages.


See also: BMAbstractController, BMAbstractMIDIController, BMMIDIBendController


Creation / Class Methods


*new (midiport, name, chan, ccArray, server)

Create a new BMMIDICCController.

midiport - An instance of BMMIDIPort.

name - A Symbol indicating the name of this controller.

chan - The MIDI Channel to receive on. Note these start from 0.

ccArray - An Array of Integers indicating which control numbers constitute this object's controls, and the order in which they should occur.

server - An instance of Server. If you pass nil this will be the default Server.

// initialise MIDI ports

BMMIDIPort.init;

// get a port

p = BMMIDIPort.ports['IAC Bus 1'];

b = BMMIDICCController.new(p, \bendController, 0, [7, 1], s); // volume and mod

b.free; // free my resources



The following three methods are useful when creating instances from GUIs.


*newFromParamDict (dict, server)

Create a new instance using a parameter dictionary. (See BMAbstractController for more information.)

dict - A Dictionary or subclass with values for the following keys: \midiport -> An instance of BMMIDIPort, \chan -> An Integer from 0 to 15, \ccArray -> An Int8Array, name -> A Symbol indicating the name of this controller.

server - An instance of Server. If you pass nil this will be the default Server.
