BMAbstractGUI abstract superclass of all GUI classes in the BEASTmulch library


Inherits from: Object


BMAbstractGUI is the abstract superclass of all GUI classes in the BEASTmulch library. It implements a few inherited methods and defines an interface.


You can see a list of all available subclasses by executing the following:


BMAbstractGUI.dumpClassSubtree;


Their help files and source code provide many useful examples.


See also: Window, View


Accessing Instance and Class Variables

name

The name displayed in the title bar of this object's Window as a String or Symbol. Usually this corresponds to the name of the object it represents.

window

Returns this object's Window.

Instance Methods


onClose_ (func)

Register a Function to be executed when this object's window is closed.

func - An instance of Function. It will be passed the receiver as an argument.



makeWindow

Subclasses should override this as needed to create a Window.



close

Close this object's window and call any function registered in onClose.
