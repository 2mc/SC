BMInOutArray an ordered Dictionary of named inputs or outputs


Inherits from: Object : Collection : SequenceableCollection : List


BMInOutArray is an ordered Dictionary of named inputs or outputs and is used to represent the input or output busses of objects in the BEASTmulch Library. Members of the collection should be either instances of Association, in the form name->busIndex, or instances of the BMSpeaker class, which mimics Association polymorphically.


In most cases members can be accessed by either key or index. For this reason numbers are not valid keys for use in this class. (Symbols are the expected key class in most places in the library in any case.)


BMInOutArray also provides support for named subarrays, specified as lists of keys.


In the current implementation no distinction is made between audio and control busses, and it is assumed that users will distinguish appropriately.


See also: BMSpeaker, Association, BMHardwareInputsProxy, Bus, List, Dictionary, BMSpeakerListVisualiser


Compatibility


BMInOutArray responds to asUGenInput,  asControlInput, asOSCArgBundle, and asOSCArgArray by returning an Array of its member's bus indices. It can thus be used directly in most places where you would pass such an Array. For example:


// make a BMInOutArray with two members

i = BMInOutArray.new;

i.add(BMSpeaker(\left, 0));

i.add(BMSpeaker(\right, 1));

// make a Synth

(

x = {|outs = #[3, 4]| 

y = SinOsc.ar([440, 660], 0, 0.2);

outs.do({|out, i| Out.ar(out, y[i])});

}.play;

)

x.set(\outs, i);

asBMInOutArray


Many classes in the BEASTMulch library which manage busses (e.g. audio sources, controllers) understand the message asBMInOutArray and return an array which corresponds to their outputs. The class Bus is extended to implement it as well as a convenience method, and takes a single name argument as the base name for its keys:


b = Bus.audio(s, 12);

a = b.asBMInOutArray(\myPrivateBusaho);


Similarly, Server is extended to include two methods: insAsBMInOutArray, and outsAsBMInOutArray. The former returns a BMHardwareInputsProxy, and the latter a BMInOutArray corresponding to the server's hardware output busses.


s.insAsBMInOutArray;

s.outsAsBMInOutArray;


Implementation Notes


BMInOutArray is a subclass of List, rather than a subclass of Dictionary although it understands most methods of the latter. Its key list is lazily constructed. This keeps the implementation simpler, and also means that BMSpeakers in the array can change their names (keys) without needing the array to be updated, but means that for very large collections it will be notably slower than say an IdentityDictionary. Since input and output collections are likely to be relatively small (i.e. < 100 elements), this should not pose a particular problem in normal usage.


Creation / Class Methods


*new (size)

Create a new BMInOutArray. In many cases you can get these directly from other objects in the library.

size - An initial capacity for the object.

// make a new BMInOutArray

i = BMInOutArray.new;

i.add(\foo->1); // add an Association

i.add(BMSpeaker(\bar, 2)); // add a BMSpeaker

i[\foobar] = 3; // C-style syntax also works

i[\foo];

i[\bar];

i[2]; // can also lookup values by index



*hardwareInputArray (server, name)

Returns an instance of BMHardwareInputsProxy corresponding to the hardware inputs for the specified server. The values of the Association pairs will be the indices of the hardware input busses.

server - The Server whose hardware inputs will be represented.

name - A String or Symbol to use as the base name. Default value is "Hardware In".

BMInOutArray.hardwareInputArray(s, 'Inputs');



*privateBusBlock (name, size, server)

Allocate a new block of private audio busses and return them as a BMInOutArray. This uses the server's normal bus allocator. The Bus objects will be stored internally so that their indices can be freed later (see freeBusObjects below).

name - A String or Symbol to use as the base name.

size - The number of busses to allocate.

server - The Server whose busses will be allocated.

p = BMInOutArray.privateBusBlock(\fooBus, 12, s);

p.freeBusObjects; // release my bus indices



Instance Methods


keys

Return this array's keys as an Array.

a = s.outsAsBMInOutArray;

a.keys;



values

Return this array's values as an Array.

a = s.outsAsBMInOutArray;

a.values;



addPrivateBusBlock (name, size, server)

Add a block of private audio busses to this array. Bus objects will be stored internally so that their indices can be freed later (see freeBusObjects below).

name - A String or Symbol to use as the base name.

size - The number of busses to allocate.

server - The Server whose busses will be allocated.

a = s.outsAsBMInOutArray.addPrivateBusBlock(\private, 10, s);

a.freeBusObjects; // release my bus indices



freeBusObjects

Free the indices for any Bus objects created using addPrivateBusBlock or *privateBusBlock. You should only do this if you don't plan to use this array and/or its elements again.



add (assoc)

Add an Association or a BMSpeaker to the end of this array.

assoc - The Association or BMSpeaker to add.

i = BMInOutArray.new;

i.add(\foo->1); // add an Association

i.add(BMSpeaker(\bar, 2)); // add a BMSpeaker



addFirst (assoc)

Insert an Association or a BMSpeaker at the start of this array.

assoc - The Association or BMSpeaker to insert.

i = BMInOutArray.new;

i.add(\foo->1); // add an Association

i.addFirst(BMSpeaker(\bar, 2)); // add a BMSpeaker before \foo



insert (index, assoc)

Insert an Association or a BMSpeaker.

index - The index at which to insert assoc.

assoc - The Association or BMSpeaker to insert.

i = BMInOutArray.new;

i.add(\foo->1); // add an Association

i.add(BMSpeaker(\bar, 2)); // add a BMSpeaker

i.insert(1, BMSpeaker(\foobar, 3)); // insert a BMSpeaker inbetween



removeAt (key)

Remove and return the value (i.e. bus index or BMSpeaker) specified by key.

key - The key of the object to be removed.

i = BMInOutArray.new;

i.add(\foo->1); // add an Association

i.add(BMSpeaker(\bar, 2)); // add a BMSpeaker

i.removeAt(\bar); // remove \bar



at (keyOrIndex)

Return the value (i.e. bus index or BMSpeaker) specified by a key or index.

keyOrIndex - The key or index of the element to be accessed

i = BMInOutArray.new;

i.add(\foo->1); // add an Association

i.add(BMSpeaker(\bar, 2)); // add a BMSpeaker

i.at(\foo); // key

i.at(1); // index

i[\foo]; // C-style syntax also works



put (key, value)

Add the value (a bus index or BMSpeaker) as an Association with key. In terms of order the Association will be added at the end of the array.

key - A Symbol to use as the key of the Association. If value is a BMSpeaker its name variable will be set to this.

value - The bus index or BMSpeaker to be added.

i = BMInOutArray.new;

i.put(\foo, 1);

i.put(\bar, BMSpeaker(\bar, 2));

i[\foobar] = 3; // C-style syntax also works



putAll (dictionaries)

Add all the elements of one or more dictionaries (usually also BMInOutArrays) to this array.

dictionaries - One or more dictionaries to be added.

i = BMInOutArray.new;

j = BMInOutArray.privateBusBlock(\foo, 3, s);

i.putAll(j, s.outsAsBMInOutArray);

j.freeBusObjects; // release my bus indices



keysValuesDo (function)

Iterate over the elements of this array evaluating function.

function - The Function to be evaluated. This will be passed the key, value, and index of each element as arguments.

i = BMInOutArray.privateBusBlock(\foo, 3, s);

i.keysValuesDo({|k, v, i| [k, v, i].postln });

i.freeBusObjects; // release my bus indices



keysDo (function)

Iterate over the elements of this array evaluating function.

function - The Function to be evaluated. This will be passed the key and index of each element as arguments.

i = BMInOutArray.privateBusBlock(\foo, 3, s);

i.keysDo({|k, i| [k, i].postln });

i.freeBusObjects; // release my bus indices

associationsDo (function)

Iterate over the elements of this array evaluating function.

function - The Function to be evaluated. This will be passed each element (an Association or a BMSpeaker) and the index as arguments.

i = BMInOutArray.privateBusBlock(\foo, 3, s);

i.put(\bar, BMSpeaker(\bar, 2));

i.associationsDo({|a, i| [a, i].postln });

i.freeBusObjects; // release my bus indices



do (function)

Iterate over the elements of this array evaluating function.

function - The Function to be evaluated. This will be passed the value and index of each element as arguments.

i = BMInOutArray.privateBusBlock(\foo, 3, s);

i.put(\bar, BMSpeaker(\bar, 2));

i.do({|v, i| [v, i].postln });

i.freeBusObjects; // release my bus indices



collect (function)

Iterate over the elements of this array evaluating function, and using the results to create a new BMInOutArray.

function - The Function to be evaluated. This will be passed the value, key, and index of each element as arguments. It should return the new value.

i = BMInOutArray.new;

i.put(\foo, BMSpeaker(\foo, 0));

i.put(\bar, BMSpeaker(\bar, 1));

i[\foo].directivity;

i = i.collect({|v, k, i| v.directivity = 'reflected' });

i[\foo].directivity;



select (function)

Iterate over the elements of this array evaluating function, adding those items for which function returns true to a new BMInOutArray.

function - The Function to be evaluated. This will be passed the value, key, and index of each element as arguments. It should return a Boolean indicating whether value should be added to the new BMInOutArray.

i = BMInOutArray.new;

i.put(\foo, BMSpeaker(\foo, 0).directivity = 'reflected');

i.put(\bar, BMSpeaker(\bar, 1).directivity = 'direct');

i = i.select({|v, k, i| v.directivity == 'direct' });



reject (function)

Iterate over the elements of this array evaluating function, adding those items for which function returns false to a new BMInOutArray.

function - The Function to be evaluated. This will be passed the value, key, and index of each element as arguments. It should return a Boolean indicating whether value should not be added to the new BMInOutArray.

i = BMInOutArray.new;

i.put(\foo, BMSpeaker(\foo, 0).directivity = 'reflected');

i.put(\bar, BMSpeaker(\bar, 1).directivity = 'direct');

i = i.reject({|v, k, i| v.directivity == 'direct' });



associationsCollectAs (function, class)

Iterate over the elements of this array evaluating function, and using the results to create a new instances of class.

function - The Function to be evaluated. This will be passed each element (an Association or a BMSpeaker) and the index as arguments.

class - The class of the new Collection.

i = BMInOutArray.new;

i.put(\foo, BMSpeaker(\foo, 0));

i.put(\bar, BMSpeaker(\bar, 1));

i.put(\foobar, 2);

j = i.associationsCollectAs({|assoc, i| assoc.key ++ " has a bus index of " ++ assoc.value.asUGenInput }, Array);



species

Returns the receiver's class, i.e. BMInOutArray.



++ (aBMInOutArray)

Concatenate the receiver with another BMInOutArray, returning a new BMInOutArray

aBMInOutArray - Explanation of aBMInOutArray. Default value is nil. Other information.

i =  s.outsAsBMInOutArray;

j = BMInOutArray.privateBusBlock(\foo, 3, s);

k = i ++ j;

j.freeBusObjects; // release my bus indices



defineSubArray (name, elementNames)

Define a subarray which can be accessed by name.

name - A Symbol indicating the name of the subarray.

elementNames - An Array of Symbols corresponding to the keys of the elements to be included in this subarray. 

i = Bus.audio(s, 12).asBMInOutArray(\myPrivateBusaho);

i.defineSubArray(\firstTwo, ['myPrivateBusaho-1', 'myPrivateBusaho-2']);

i.getSubArray(\firstTwo);



getSubArray (name)

Returns a new BMInOutArray corresponding to the subarray specified by name.

name - A Symbol indicating the subarray to retrieve.

i = Bus.audio(s, 12).asBMInOutArray(\myPrivateBusaho);

i.defineSubArray(\firstTwo, ['myPrivateBusaho-1', 'myPrivateBusaho-2']);

i.getSubArray(\firstTwo);



getSubArrayKeys (name)

Returns an Array containing the keys of the elements contained in the subarray specified by name.

name - A Symbol indicating the subarray whose keys will be retrieved.

i = Bus.audio(s, 12).asBMInOutArray(\myPrivateBusaho);

i.defineSubArray(\firstTwo, ['myPrivateBusaho-1', 'myPrivateBusaho-2']);

i.getSubArrayKeys(\firstTwo);



removeSubArray (name)

Remove an existing subarray from the list of subarrays. (The elements themselves remain in the array).

name - A Symbol indicating the subarray to remove.

i = Bus.audio(s, 12).asBMInOutArray(\myPrivateBusaho);

i.defineSubArray(\firstTwo, ['myPrivateBusaho-1', 'myPrivateBusaho-2']);

i.subArrays;

i.removeSubArray(\firstTwo);

i.subArrays;



addToSubArray (name, element)

Add an element to an existing subarray.

name - A Symbol indicating the subarray to which the element will be added.

element - A Symbol corresponding to the key of the element to be added.

i = Bus.audio(s, 12).asBMInOutArray(\myPrivateBusaho);

i.defineSubArray(\firstTwo, ['myPrivateBusaho-1', 'myPrivateBusaho-2']);

i.addToSubArray(\firstTwo, 'myPrivateBusaho-3');

i.getSubArrayKeys(\firstTwo);



removeFromSubArray (name, element)

Remove an element from an existing subarray.

name - A Symbol indicating the subarray from which the element will be removed.

element - A Symbol corresponding to the key of the element to be removed.

i = Bus.audio(s, 12).asBMInOutArray(\myPrivateBusaho);

i.defineSubArray(\firstTwo, ['myPrivateBusaho-1', 'myPrivateBusaho-2']);

i.removeFromSubArray(\firstTwo, 'myPrivateBusaho-1');

i.getSubArrayKeys(\firstTwo);



subArrays

Returns an Array of Symbols corresponding to the names of all defined subarrays.

i = Bus.audio(s, 12).asBMInOutArray(\myPrivateBusaho);

i.defineSubArray(\firstTwo, ['myPrivateBusaho-1', 'myPrivateBusaho-2']);

i.defineSubArray(\threeAndFour, ['myPrivateBusaho-3', 'myPrivateBusaho-4']);

i.subArrays;



boundaries

Iterate over the array and calculate the smallest rectangular cuboid (aligned with the axes) which can contain all the BMSpeakers in the list. Associations are ignored. Returns an Array of two Arrays in the form [[minX, minY, minZ], [maxX, maxY, maxZ]]

isBMInOutArray

Returns true. This method is defined in an extension to Object to return false and can thus be used for testing.
