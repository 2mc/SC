BMHardwareInputsProxy represents the hardware inputs on a server


Inherits from: Object : Collection : SequenceableCollection : List : BMInOutArray


This subclass of BMInOutArray represents the hardware input busses of a server. Normally you create instances by calling BMInOutArray.hardwareInputArray. It allows the hardware inputs to be represented in visualisations of an audio chain.
