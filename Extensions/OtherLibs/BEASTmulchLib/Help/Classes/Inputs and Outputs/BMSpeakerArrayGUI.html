BMSpeakerArrayGUI a GUI for creating a speaker array


Inherits from: Object : BMAbstractGUI


BMSpeakerArrayGUI provides a GUI for creating a speaker array. It provides all existing BMSpeakerSpecs as a draggable list, and allows users to specify subarrays.


N.B. In the GUI 'Hardware Outputs' start at 1, not 0. They will be converted to the correct SC bus numbers when the speakers are created.


See also: BMAbstractGUI, BMSpeaker, BMInOutArray, BMSpeakerSpec


Creation / Class Methods


*new (startArray, okayFunc, name, origin)

Create a new BMSpeakerArrayGUI.

startArray - An optional instance of BMInOutArray to serve as a starting point.

okayFunc - A Function that will be evaluated when the window is closed. This will be passed the resulting BMInOutArray as an argument.

name - An optional String or Symbol indicating the title of the window. The default is "Define Speaker Array".

origin - An optional Point specifying the origin of the window, x@y.



Examples


BMSpeakerArrayGUI.new(nil, {|array| array.postln });
