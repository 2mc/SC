BMSelectInsOutsGUI a GUI for selecting inputs and outputs


Inherits from: Object : BMAbstractGUI


BMSelectInsOutsGUI is a modal sheet which allows you to select subsets of inputs and outputs from supplied lists.


See also: SCModalSheet, BMInOutArray


Creation / Class Methods


*new (parent, ins, outs, okFunc)

Create a new BMSelectInsOutsGUI.

parent - The parent window to which this sheet will be attached.

ins - A BMInOutArray.

outs - A BMInOutArray.

okFunc - A Function that will be evaluated when the sheet is closed. This will be passed two BMInOutArrays, inResults and outResults, as arguments.


Examples


(

w = Window("Dummy Parent", Rect(400, 400, 400, 400)).front; // a blank window to attach this to

BMSelectInsOutsGUI(w, s.insAsBMInOutArray, s.outsAsBMInOutArray, {|ins, outs|

"ins %\n".postf(ins);

"outs %\n".postf(outs);

});

)
