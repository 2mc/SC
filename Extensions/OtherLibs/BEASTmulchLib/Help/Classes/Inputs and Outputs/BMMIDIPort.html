BMMIDIPort represents a MIDI connection


Inherits from: Object


BMMIDIPort represents a single MIDI connection.


See also: BMAbstractMIDIController, MIDI, MIDIIn, MIDIOut


Double-sided vs. Single-sided Ports


When *init is called it constructs an IdentityDictionary of all available ports. If a source has a matching destination this method will combine them into a single port, otherwise you can have ports with only a source or only a destination (i.e. only input or output).


Creation / Class Methods


*init

This constructs the list of available ports, calling MIDIIn.connectAll if MIDIClient has not been initialized.



*new (name, inuid, outuid, device, outport)

Create a new BMMIDIPort. Normally you would not do this directly; rather it is done automatically when you call *init.

name - A String corresponding to the name of this port.

inuid - The inuid. This should be the same as the corresponding source MIDIEndPoint.

outuid - The outuid. This should be the same as the corresponding destination MIDIEndPoint.

device - A String corresponding to the name of this port's device. This should be the same as the corresponding source MIDIEndPoint(s).

outport - The index of this port's destination MIDIEndPoint in MIDIClient.destinations.



Accessing Instance and Class Variables

*ports

If *init has been called this will return an IdentityDictionary containing all available ports. Keys are Symbols corresponding to each port's name.

*initialised

Returns a Boolean indicating whether or not *init has been called.

inuid

This port's inuid. This should be the same as the corresponding source MIDIEndPoint.

outuid

This port's outuid. This should be the same as the corresponding destination MIDIEndPoint.

device

Returns a String indicating the name of this port's device.

outport

The index of this port's destination MIDIEndPoint in MIDIClient.destinations.

name

Returns a Symbol indicating the name of this port.

Instance Methods

initFromArchive

Checks to see if *init has already been called, and if not, posts a warning.
